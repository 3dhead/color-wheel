## Color Wheel [WIP]

Color picker React component

```js
import ColorWheel from 'color-wheel';

// usage:
<ColorWheel
	onChange={color => console.log(color)}
/>
```

### Develop

```
cd demo
npm run serve
npx run build -w
```
